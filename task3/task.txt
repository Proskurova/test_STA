Создать следующие программы:
1) Программа (file_generation.py), которая каждые 5 секунд генерирует текстовый файл с произвольным именем, 
содержание файла = имя файла, в папке data.
2) Программа (publisher.py), которая каждые 15 секунд будет отправлять текущий список файлов папки data 
(в любом формате, например json) в RabbitMq.
3) Программа (consumer.py), которая читает сообщения из RabbitMq. Сообщения читаются сразу, как publisher 
их отправит в RabbitMq. Полученное сообщение обрабатывается 35 секунд (имитация обработки файлов), а именно:
 содержимое фалов объединяется и создается текстовый файл в папке result. Имя файла = идентификатор сообщения RabbitMQ.
 Исходные файлы удаляет.
Программа file_generation должна работать две минуты.
Для работы с RabbitMQ использовать библиотеку pika.
